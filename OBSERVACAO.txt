TIPOS DE DADOS!!!

- int ( inteiros)

- char (letras, lembrando que para colocar a quantidade de letras que vc quer precisa estar entre [] )

- float (numeros com casa decimais ou fracionarios)

-double ( float 2 com precisáo dupla)

-enum ( dados enumerados posibilitam os tipos definidos pelo usuario)

-bool ( true or false)

-void ( valores que ocupam 0 bits e nao possuem valor, ausencia de tipo)

-pointer (endereco de memoria)

#define <nome> e o valor ( CONSTANTE, VARIAVEL QUE NAO MUDA O VALOR)

MODIFICADORES DE TIPOS DE DADOS:

- unsigned (declarar que um valor numerico nao tera sinal)
- short (reduz a capacidade de armazenamento)
- long (aumenta a capacidade de armazenamento)

exemplos : - long int , - unsigned float
